# Pre-Workshop Requirements

Participants are expected to arrive at the bootcamp with a working system. Please complete the following **before** March 11, 2026.

## Required

1. **Get a Claude Code subscription**
   - A Pro subscription should be sufficient for this bootcamp.
   - You may need to figure out who will pay for it -- talk to your PI if unsure.

2. **Install Claude Code**
   - See the [official installation guide](https://docs.anthropic.com/en/docs/claude-code/overview) for instructions.
   - Verify it works by typing `claude` in any terminal window.

## Highly Recommended

3. **Install VSCode**
   - Download from [code.visualstudio.com](https://code.visualstudio.com/).

4. **Install GitHub Desktop**
   - Download from [desktop.github.com](https://desktop.github.com/).

5. **Clone the bootcamp repo**
   - https://github.com/matentzn/aicuration_workshop

6. **Clone the Mondo repo**
   - We MIGHT use real Mondo tickets for practice exercises.

7. **Clone any repos of personal interest**
   - If you have specific curation repos you work with (e.g. VBO, ECTO, ZP), have them cloned and ready.
