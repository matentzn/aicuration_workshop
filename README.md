# AI4Curation Bootcamp - Workshop Planning

This repository contains planning materials for the **AI4Curation Bootcamp**, the first AI-for-curation workshop for the Monarch Initiative. The workshop runs March 11-13, 2026 (3 hours/day) and teaches Monarch curators (non-developers) how to use Claude Code in their curation workflows.

## What This Repo Is For

This is a **workshop planning repo**. It is used to organise, develop, and refine the agenda, exercises, and supporting materials for the bootcamp. When asked to work on this repo, you are helping plan and prepare the workshop -- not building software.

## Files

| File | Purpose |
|---|---|
| `README.md` | This file. Overview of the project and file inventory. |
| `CLAUDE.md` | Project-level instructions for Claude Code when working in this repo. |
| `AGENDA.md` | The cleaned-up workshop agenda with rough timings for all 3 days. This is the primary deliverable being iterated on. |
| `REQUIREMENTS.md` | Pre-workshop requirements for participants -- what to install and set up before arriving. |
| `background/Monarch_ AI4Curation training (2026).pdf` | Original planning document with raw notes, brainstormed topics, participant lists, and requirements. Source of truth for what was discussed in planning meetings. |

## Workshop Structure (Summary)

- **Day 1:** Clean House -- Claude Code setup, CLAUDE.md files, settings, security, basic task workflows
- **Day 2:** Agents and Skills -- using, finding, and creating skills; SOPs for Monarch
- **Day 3:** Advanced Workflows -- validation skills, complex curation tasks (mappings, spreadsheets), repo setup for success
